FreePalp

Multi-agent AI система с самокоррекцией, ReAct loop и автономным использованием инструментов. Агент сам планирует задачи, вызывает инструменты, проверяет качество и при необходимости повторяет — без участия пользователя.

Ключевые системы

ReAct Loop ключевое

Worker не просто генерирует текст — автономно вызывает инструменты, получает данные, сохраняет результаты и формирует финальный ответ.

Live Model Discovery

Автоматически обнаруживает доступные модели у 10+ провайдеров в реальном времени. Роутер выбирает оптимальную под задачу.

Self-Improvement уникально

Система может анализировать свой собственный код через read_source/write_source и вносить улучшения. 15+ версий промптов в архиве.

Vector Memory

Персистентная векторная память с горячим и тёплым слоями, архивом коррекций и профилем пользователя.

41 инструмент из коробки

Файлы, браузер, GitHub, shell, веб-поиск, уведомления, системный мониторинг — агент вызывает их автономно.

MCP Discovery

Автоматически обнаруживает и подключается к MCP-серверам для расширения набора инструментов.

Cron Manager

Планировщик задач — агент может выполнять задачи по расписанию без участия пользователя.

Skill Distillation уникально

Удачные коррекции дистиллируются в переиспользуемые документы SKILL.md — провалы дешёвых моделей превращаются в навык для будущих задач.

OpenAI-совместимый API

Работает как OpenAI-эндпоинт — подключается к IDE-плагинам (Continue.dev и др.). Critic двухуровневый: сначала детерминированные проверки, затем LLM-оценка.
