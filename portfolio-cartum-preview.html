Cartum Bridge Suite

Cartum Control Center

Настольная утилита автоматизации деплоя для платформы Cartum / Horoshop. Заменяет ручное копирование CSS, JS и HTML в редактор платформы — одной кнопкой загружает все файлы, сохраняет авторизацию и делает резервные копии.

Python Tkinter GUI Playwright Cartum / Horoshop Автоматизация RSA шифрование Node.js
О проекте

Cartum Control Center — внутренний инструмент, созданный для устранения рутины при работе с темами интернет-магазинов на платформе Cartum / Horoshop. Стандартный процесс требовал вручную открывать браузер, авторизоваться в админке и по одному вставлять CSS и HTML-фрагменты в редактор.

Программа автоматизирует весь цикл: выбор папки с исходниками → авторизация → копирование в src → тихая загрузка в платформу. Всё это в одном окне.

Ключевые системы

Авторизация с сохранением сессии

Вход в админку Cartum через Playwright один раз — сессия сохраняется в .auth/cartum.json. Повторная авторизация не нужна до истечения сессии.

Тихий деплой одной кнопкой

Копирует desktop.css, mobile.css, HTML-фрагменты в папку src и загружает напрямую в Cartum без открытия браузера вручную.

Система резервных копий

Автоматические бэкапы перед каждым деплоем. Политики хранения: 1 / 7 / 30 дней или всегда. Восстановление любой версии в один клик из вкладки «Сохранения».

ZIP-источник уникально

Если в папке-источнике найден ZIP-архив с нужным префиксом — файлы берутся из него автоматически, без ручной распаковки.

Зашифрованная телеметрия уникально

Аналитика использования шифруется RSA публичным ключом. Расшифровать данные может только владелец через отдельную owner-консоль с приватным ключом. Отправка только по явному согласию пользователя.

UDP Discovery owner-консоли уникально

Автоматически обнаруживает relay-консоль в локальной сети по UDP на порту 8766 — не нужно вручную указывать IP.

Система миграции конфигурации

При обновлении до новой версии автоматически переносит все настройки, пути и авторизацию из старой установки.
