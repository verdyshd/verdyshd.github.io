AI Freelance Agent FAS

MVP системы автоматической обработки фриланс-заявок. Телеграм-бот принимает описание вакансии, AI-агент анализирует, оценивает и генерирует отклик — пользователь только одобряет или редактирует результат.

Интерфейс
Как работает
1. Пользователь вводит /newjob → вставляет текст вакансии
2. Normalizer нормализует и структурирует данные
3. Scorer оценивает совместимость (0–100)
4. Strategy выбирает тактику отклика
5. Writer Agent генерирует персонализированный отклик
6. Critic Agent проверяет и улучшает качество
7. Бот показывает результат с кнопками: одобрить / редактировать / пропустить
Ключевые системы

Telegram Bot UI

Полноценный интерфейс в Telegram с inline-кнопками, FSM состояниями и форматированными сообщениями. Команды: /start, /newjob, /jobs, /profile, /mode.

Writer + Critic Pipeline

Двухэтапная генерация: Writer создаёт отклик, Critic оценивает и предлагает улучшения. Итеративная доработка до приемлемого качества.

Scoring Engine

Движок оценки совместимости вакансии с профилем пользователя. Учитывает навыки, опыт, предпочтения по ставке и типу занятости.

SQLite + SQLAlchemy

Постоянное хранение вакансий, откликов, профиля и истории. Полноценные ORM-модели с репозиторием.

template (без AI), openai, ollama (локально), manual_chat (промпт готовится ботом, ты вставляешь в любой чат AI).
