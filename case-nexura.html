О проекте

Nexura — нишевое контент-издание: практичные гайды и сравнения техники для рынка Молдовы на двух языках (RU/RO), с отдельной презентационной EN-веткой. Каждая статья проходит редакционную проверку: источники, дата ревью, локальные условия Молдовы, запрет на фиктивные «тесты». Сайт — MVP для проверки бизнес-гипотезы по нише.

Строгий редакционный пайплайн

Публикация только с источниками (≥2, включая первичный), датой ревью, локальной привязкой к Молдове и запретом необоснованных тестов.

Production-аудит авто

Скрипт грузит каждый URL из живого sitemap и проверяет HTTP, canonical, языковые альтернаты, Open Graph, структурированные данные, аналитику и robots.txt.

Lighthouse 99 / 100 / 100 / 100

Performance 99, Accessibility, Best Practices и SEO — по 100. Проверено на desktop и mobile.

CI и авто-деплой

GitHub Actions собирает сайт и публикует на Cloudflare Pages. Подключены Web Analytics и Google Search Console, sitemap отправлен.

Семантика и кластеры

108 запросов RU/RO прогнаны через Keyword Planner (регион Молдова), ручной разбор SERP. Выбраны 3 кластера: робот-пылесосы, AI для работы, умный дом.

Статус

Инженерно готов и опубликован. Цель ещё не закрыта: бизнес-гипотеза проверяется внешними данными — наблюдение Search Console 8–12 недель (индексация, первые показы, реальные запросы и CTR) и числа Keyword Planner, недоступные без платной кампании. До появления данных — никакой массовой генерации статей и монетизации.
