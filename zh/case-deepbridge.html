DeepBridge

DeepBridge 是 AIOSE（AI OS Environment）的当前可用原型 —— 一个我长期打造的「面向 AI 的环境」：浏览器版 DeepSeek 与带工具的完整 AI 智能体之间的本地桥接。早期的 DeepGate 因思路有误而弃用 —— 由更稳定的 DeepBridge 取代：通过浏览器拦截与注入请求、会话管理、Web UI 中的文件系统与 shell 工具。

关于项目

DeepBridge 是一座本地桥接，将浏览器版 DeepSeek 变为可访问文件系统、终端与浏览器的完整 AI 智能体。Flask 服务本地运行，Playwright 通过 CDP 控制 Chrome，Web UI 提供 DeepSeek 风格、支持工具的界面。

思路演进

AIOSE 系列

AI OS Environment —— 整个系列的总称，一个早已开始打造的「面向 AI 的环境」。DeepGate 与 DeepBridge 是其中的迭代。

DeepGate 已弃

最初的拦截方案。被证明不稳定且架构有误 —— 已停止开发。

DeepBridge 当前

重新设计的桥接：会话与注入更稳定。可用原型，仍需完善。

核心系统

CDP Bridge 独特

通过 Chrome DevTools Protocol 控制 Chrome，无需官方 API 即可拦截并注入对 DeepSeek 的请求。

File System Tools

智能体在工作目录中读写与删除文件。防护：路径穿越校验、目录白名单、受保护文件。

Shell Tools

通过智能体执行终端命令。命令白名单校验、捕获 stdout/stderr、防止破坏性操作。

Session Management

聊天会话管理：历史、导出、右键菜单与 token 统计。完整的对话生命周期。

DeepSeek 风格 Web UI

极简界面：侧栏、模式（Think / Normal）、快捷键、主题与历史搜索。20+ UI 测试截图。

令牌鉴权

重启时从文件加载持久访问令牌。防止对本地服务器的未授权访问。
