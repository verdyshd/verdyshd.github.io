AI Freelance Agent FAS

自动处理自由职业需求的 MVP。Telegram 机器人接收职位描述，AI 智能体分析、评分并生成回复 —— 用户只需批准或编辑结果。

界面
工作原理
1. 用户输入 /newjob → 粘贴职位文本
2. Normalizer 清洗并结构化数据
3. Scorer 评估匹配度（0–100）
4. Strategy 选择回复策略
5. Writer Agent 生成个性化回复
6. Critic Agent 校验并提升质量
7. 机器人展示结果并提供按钮：批准 / 编辑 / 跳过
核心系统

Telegram Bot UI

完整的 Telegram 界面：内联按钮、FSM 状态与格式化消息。命令：/start、/newjob、/jobs、/profile、/mode。

Writer + Critic Pipeline

两阶段生成：Writer 起草回复，Critic 评分并提出改进。迭代优化至可接受质量。

Scoring Engine

评估职位与用户档案匹配度的引擎。考虑技能、经验、费率与用工类型偏好。

SQLite + SQLAlchemy

持久化存储职位、回复、档案与历史。完整的 ORM 模型与仓储层。

4 种 AI 模式

template（无 AI）、openai、ollama（本地）、manual_chat（机器人准备提示词，你粘贴到任意 AI 聊天）。
